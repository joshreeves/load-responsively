# load-responsively.js
A jQuery plugin that uses the aspect ratio of not-yet-loaded images to reserve responsive space.

### Usage
```js
$(document).ready(function() {
  $("img").loadResponsively();
});
```

### Browser support
Compatible with All Browsers and IE7+.
